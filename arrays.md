# arrays

**Вопрос 1**
Сколько элементов имеет следующий массив?
1int[,,] numbers = new int[3, 2, 3];
Варианты ответов
1) 0 
2) 1
3) 8
4) 9
5) 11
6) 18
Ответ: 6) 18

**Вопрос 2**
Сколько измерений (размерность) имеет следующий массив?
1int[,] numbers = new int[3, 3];
Варианты ответов
1) 1
2) 2
3) 3
4) 6
Ответ: 2) 2

**Вопрос 3**
Что будет выведено на консоль в результате выполнения следующего кода:
```cs
int[][] nums = new int[3][];
nums[0] = new int[2] { 1, 2 };
nums[1] = new int[3] { 3, 4, 5 };
nums[2] = new int[5] { 6, 7, 8, 9, 10 };

Console.WriteLine(nums[3][2]);
```
**Ответ:** Будет сгенерировано исключение **`System.IndexOutOfRangeException`** (Исключение выхода за пределы диапазона индексов), и программа сразу завершится.

**Вопрос 4**
Дан следующий массив
```cs
int[][] nums = new int[3][];
nums[0] = new int[2] { 1, 2 };
nums[1] = new int[3] { 3, 4, 5 };
nums[2] = new int[5] { 6, 7, 8, 9, 10 };
```
Каким образом мы можем обратиться к числу 7 в этом массиве?
**Ответ:** `nums[2][1]`


**Практическая**
Задан следующий трехмерный массив:

int[,,] mas = { { { 1, 2 },{ 3, 4 } },
	{ { 4, 5 }, { 6, 7 } },
	{ { 7, 8 }, { 9, 10 } },
	{ { 10, 11 }, { 12, 13 } }
	};

С помощью циклов переберите все элементы этого массива и выведите их на консоль в
следующем виде:
{{{1 , 2} , {3 , 4}} , {{4 , 5} , {6 , 7}} , {{7 , 8}, {9 , 10}} , {{10 , 11} , {12 , 13}}}


```cs
using System;

class Program
{
    static void Main()
    {
        int[,,] mas = {
            { { 1, 2 }, { 3, 4 } },
            { { 4, 5 }, { 6, 7 } },
            { { 7, 8 }, { 9, 10 } },
            { { 10, 11 }, { 12, 13 } }
        };

        Console.Write("{");
        for (int i = 0; i < mas.GetLength(0); i++)
        {
            Console.Write("{");
            for (int j = 0; j < mas.GetLength(1); j++)
            {
                Console.Write("{");
                for (int k = 0; k < mas.GetLength(2); k++)
                {
                    Console.Write(mas[i, j, k]);
                    if (k < mas.GetLength(2) - 1) Console.Write(" , ");
                }
                Console.Write("}");
                if (j < mas.GetLength(1) - 1) Console.Write(" , ");
            }
            Console.Write("}");
            if (i < mas.GetLength(0) - 1) Console.Write(" , ");
        }
        Console.WriteLine("}");
    }
}
```
